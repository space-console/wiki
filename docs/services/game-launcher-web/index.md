# game-launcher-web

The **TV / screen** of Space Console: an AirConsole-style party-game launcher
built for the 10-foot living-room UI (webOS, Android TV, tvOS) and desktop
browsers. Phones join as controllers (see
[game-controller](../game-controller/index.md)).

It is a **zero-build, zero-backend static site** — plain ES modules, no bundler,
no framework, no compile step. Open `index.html` and it runs.

- [Getting started](getting-started.md) — run it locally.
- [Architecture](architecture.md) — modules, the input-intent layer, and the seams.

Production: <https://space-console.github.io/game-launcher-web/>

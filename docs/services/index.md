# Services

Per-service documentation. The full index — roles, production URLs, preview
pattern — lives in [Services & repositories](../repositories.md).

- [game-launcher-web](game-launcher-web/index.md) — the TV / screen (launcher).
- [game-controller](game-controller/index.md) — the phone / controller client.
- [games](games/index.md) — the collection of playable games (30 of them).
- [tetris](tetris/index.md) — the first playable game; runs on the launcher.
- [web-api](web-api/index.md) — the WebRTC signaling relay (TV↔phone pairing).

All services share one build/deploy pipeline; see
[Way of working](../way-of-working.md).

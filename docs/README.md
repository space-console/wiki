# Documentation

Org-wide documentation for **Space Console** — cross-project guides and reference material that doesn't belong to any single repo.

This wiki is also the **process reference for AI agents** working in the org —
the [Way of working](way-of-working.md) page is the source of truth for how we
build, preview, review, and ship across every repo.
